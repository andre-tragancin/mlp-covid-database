# How to install dependencies to Project

*Note*: This guide was made using Python 3.8 and Windows 10

## Install OpenCv

```
pip install opencv-python
```

## Install skimage

```
python -m pip install -U scikit-im age
```

## Install sklearn

```
pip install -U scikit-learn
```

## Install numpy

```
pip install numpy
```
